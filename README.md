# parkshark
Helps you find parking a little more efficiently

## Requirements

### Node Packages
- Mongoose >= 6.5.0
- Express >= 4.18.2
- Firebase >= 5.2.9
- React == 18.1.0
(Refer to package.json for additional packages)

### Backend 
- NodeJS
- Firebase Email/Password Auth
- MongoDB


## Start backend 
Set `MONGODB_URL=mongodb+srv://<user>:<password>@<URL>/?retryWrites=true&w=majority`
`$ npm run serve`

## Start frontend
'$ npm start'
